About the app

Barcodehammer — read and create barcodes, offline

About Privacy policy Support Terms of use Language: Nederlands · English · Español

Barcodehammer reads and creates barcodes and QR codes. Fully offline: no internet connection, no account and no advertising. What you scan or create stays encrypted on your own device.

What the app does

  • Scan — point the camera at a code and read it.
  • Create — pick a format, type the content and the app draws the code.
  • History — everything you scan and create, stored encrypted, removable per entry or all at once.

Which kinds of code

Read and create (13): QR, Data Matrix, Aztec, PDF417, EAN-13, EAN-8, UPC-A, UPC-E, Code 128, Code 39, Code 93, ITF and Codabar.

Create only (7): EAN-2, EAN-5, ISBN, GS1-128, ITF-14, Telepen and RM4SCC.

Which kinds can be read is determined by the operating system's barcode recognition; the app cannot add to that list. For a format you cannot scan back, the app says so before you create it.

Check digits

For EAN-13, EAN-8, UPC-A, UPC-E, ISBN and ITF-14 the app works out the last digit itself if you leave it off, and tells you which digit was added. If you do type it, the app verifies it. Length, permitted characters and capacity are checked before anything is drawn — so you get an explanation instead of a code no scanner can read.

Privacy in one sentence

The app asks only for the camera, and only to read codes. No photos are taken or stored. See the privacy policy.
