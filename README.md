Statische HTML, geen buildstap. `.nojekyll` zorgt dat GitHub Pages de bestanden
ongewijzigd serveert.

## Herkomst

Deze repository bevat **alleen de website**. De broncode van de app staat in een
aparte, private repository. Bewerk de pagina's daar in de map `docs/` en
publiceer ze hierheen met `publiceer-website.ps1`; wijzigingen die je alleen
hier maakt, raak je bij de volgende publicatie kwijt.
