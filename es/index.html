Sobre la aplicación

Barcodehammer — leer y crear códigos de barras, sin conexión

Sobre la app Política de privacidad Asistencia Condiciones de uso Idioma: Nederlands · English · Español

Barcodehammer lee y crea códigos de barras y QR. Totalmente sin conexión: sin internet, sin cuenta y sin anuncios. Lo que escaneas o creas permanece cifrado en tu propio dispositivo.

Qué hace la aplicación

  • Escanear — apunta la cámara a un código y léelo.
  • Crear — elige un formato, escribe el contenido y la aplicación dibuja el código.
  • Historial — todo lo que escaneas y creas, guardado cifrado, se puede borrar por entrada o todo de una vez.

Qué tipos de código

Leer y crear (13): QR, Data Matrix, Aztec, PDF417, EAN-13, EAN-8, UPC-A, UPC-E, Code 128, Code 39, Code 93, ITF y Codabar.

Solo crear (7): EAN-2, EAN-5, ISBN, GS1-128, ITF-14, Telepen y RM4SCC.

Qué tipos se pueden leer lo determina el reconocimiento de códigos del sistema operativo; la aplicación no puede ampliar esa lista. Si un formato no se puede volver a escanear, la aplicación lo indica antes de crearlo.

Dígitos de control

En EAN-13, EAN-8, UPC-A, UPC-E, ISBN e ITF-14 la aplicación calcula ella misma el último dígito si lo omites, e indica cuál ha añadido. Si lo escribes, comprueba si es correcto. También verifica la longitud, los caracteres permitidos y la capacidad antes de dibujar nada, de modo que recibas una explicación en lugar de un código que ningún lector puede leer.

Privacidad en una frase

La aplicación solo pide la cámara, y únicamente para leer códigos. No se toman ni se guardan fotos. Consulta la política de privacidad.
